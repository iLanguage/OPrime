OPrime

OPrime is an OpenSource Experimentation library which permits data collection of eye-gaze, audio, video and touch on Android tablets, phones and anywhere Chrome runs (Mac, Linux, Windows, iPad, Android, Chromebooks). The project integrates a number of popular OpenSource libraries for data analysis and machine learning, including OpenCV, FFMpeg, Sphinx, HTK, Praat, Git, Node.js, Bootstrap, Angular.js among others.

Android

Android tablets use a BlackBoard expert system architecture which permits modules and apps to work together to build a more complex system. This reduces the time which it takes to develop a powerful experiment or crowd sourcing data collection pipline.

ChromeApps

Chrome is the fastest moving browser, implementing new features every week and building more and more integration with hardware (ie microphones, cameras and file system) which permits running experiements using the flexablity of a browser, but offline in remote locations or basements. Applications can be deployed in the Chrome Store and run offline with unlimited space for data.

Node.js

Node.js is a very flexible server written in Javascript. It is capbable of integrating closely with any software which provides a command line interface. We use Node.js to automate the data extraction and analysis pipelines through a number of OpenSource libaraires specialized in audio, video or language processing.

Any Server

Because almost all of the HTML5 logic is written in Javascript, and runs client side, OPrime experiments can be run from any webserver, from any web hosting company or from any university web site.
